# Rundeck Custom Notification Plugin

This notification plugin allows you send customized messages to a webhook using several parameters.  


## Installation

- Download the plugin and put the file into the rundeckÂ´s libext folder.


## Usage

### Properties setting

- **_Webhook URL_**: Valid callback url or HTTP Endpoint

- **_Content Type_**: Message content type

- **_Request Method_**: The request method

- **_Message Body_**: The message body
